# Immune_globin
The code used for data analyses performed in R and linux that was used to produce figures presented in the article.

The analysis includes two parts:

1. RNA-seq analysis of sorted Ter119+ cells from the fetal liver of E12.5 and E14.5 embryos (FL12.5 and FL14.5) and the bone marrow of 2-week-old and 8-week-old mice (BM2W and BM8W).
2. RNA-seq, ATAC-seq and CUT&Tag analysis of H3K9ac/H3K27ac of IRF1-overexpressed human primary erythroblasts.
